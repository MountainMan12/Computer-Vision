# Image classification using deep learning
### Libraries: keras, matplotlib, sys
### Language: Python 3.7.4
### Dataset: CIFAR-10

### Paramteres: 
1) Learning rate = 0.001
2) momentum = 0.9
3) Loss function = Categorical cross entropy
4) epochs = 100
5) batch size = 64
            
Validation accuracy of the model = 84.6%
